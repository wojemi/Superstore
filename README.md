# Superstore
This repository contains SQL code describing and analyzing the data concerns 'Superstore' using SQL and Power BI - the store with different kind of articles. 

# Business problem
Retail companies often face situations where their profits are unstable or declining even when sales are increasing. This aim of this prject is to analyze the condition of such companies and  identify factors that may affect their profitability.

# Dataset
Dataset used for this project comes from the KajoData SQL course and contains information about the store's transactions, products and customers.
Dataset includes:
- order date,
- customer id
- state
- product name
- sales
- profit
- discount etc.

# Tools
- SQL - main analysis
- Microsoft Power BI - dashboard and data visuazlization
- Excel - preliminary data exploration

# Project structure
1. Data - raw dataset
2. SQL - analyzing and drawing conclusions
3. Power BI - dashboard

# Business questions
1. How discounts affect sales?
2. How sales and profits changed over time (2014-2017)?
3. Why in 2015 profit decreased even if sales increased?
4. In which months of 2014 sales were the highest and the lowest?
5. How sales changed QoQin 2014-2017?
6. Which products generate the highest profits and which one the largest losses?
7. How many customers are responsible for 75% of sales?
8. Which state generates the highest sales and profit?

# Key insights
1. Discounts contribute to increased sales.
2. In 2015 profits fell compared to 2014.
3. The margin and discount increasing are the reason for the lower profit in 2015.
4. First quarters of years 2014-2017 are characterized by a decline in sales.
5. Furniture category generates the highest sales and the lowest profits.

# Business recommendations
1. Focus on selling products that generates the highest profits.
2. Monitoring the level of discounts that may reduce profits.
3. Marketing campaigns that will help increase sales in first quarters of the analyzed period.
